# Training Planner (План тренировок)

**Автор:** Абдракипов Айрат

## Описание
GUI-приложение для планирования и учёта тренировок. Позволяет добавлять записи с датой, типом тренировки и длительностью, выполнять фильтрацию по дате и типу, а также сохранять данные в JSON-файл.

## Основные возможности
- Добавление тренировок через удобную форму
- Валидация ввода: дата должна быть в формате YYYY-MM-DD, длительность — положительное целое число, тип — не пустая строка
- Автоматическое сохранение данных в trainings.json
- Фильтрация таблицы по дате, типу тренировки или их комбинации
- Сброс фильтров
